# 📧 Spam Email Detection using Machine Learning

This project builds a machine learning model to classify emails as **Spam** or **Not Spam** using Logistic Regression.

---

## 🚀 Features
- Data preprocessing and feature scaling
- Logistic Regression model training
- Achieved **91.97% accuracy**
- Confusion matrix visualization
- Random email prediction

---

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

---

## 📊 Dataset
- Spam email dataset (CSV format)
- Contains labeled emails (Spam = 1, Not Spam = 0)

---
